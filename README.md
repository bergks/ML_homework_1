# ML_homework_1
## Датасет
**Mushroom Classification (Kaggle)**
- 22 признака, 8124 образца
- Задача: бинарная классификация (съедобный/ядовитый)
- Целевая метрика: recall по ядовитым грибам (не хотим пропустить опасный)

Выбрала, потому что было интресно поработать с бинарной классификацией. Также потому что признаки казались понятными для обывателя

## Гипотезы
1. Дерево решений покажет более высокий recall, чем логистическая регрессия с one-hot кодированием, так как все признаки категориальные
2. Объединение редких категорий в 'other' снизит переобучение логистической регрессии и повысит recall на тесте
3. В данных есть супер-предиктор, по которому можно классифицировать грибы с высокой точностью
4. Baseline модель только по признаку 'odor' даст >95% метрик
5. Дерево решений достигнет идеальных метрик при небольшой глубине, дальше подбирать параметры бессмысленно

## Результаты
1. Подтвердилась. Дерево дало recall 1.0 против 0.996 у LogReg
2. Подтвердилась. Объединение редких категорий не ухудшило recall (остался 0.996), но сократило число признаков
3. Подтвердилась. Признак odor оказался супер-предиктором
4. Подтвердилась. Baseline по одному odor дает 96.9% recall.
5. Подтвердилась. Дерево вышло на идеал при глубине 7, дальше подбор не нужен

## Вывод
Задача оказалась очень простой из-за наличия супер-предиктора odor, поэтому методы машинного обучения не дают преимущества перед простым baseline по этому признаку. Однако, с помощью дерева решений можно достичь идеальной точности
